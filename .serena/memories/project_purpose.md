# vscode-edge-devtools Project Purpose

A lightweight browser preview extension for Visual Studio Code that lets you view and interact with web applications directly inside your editor using Microsoft Edge.

## Key Features
- Live Browser Preview
- Multiple Browser Instances
- Device Emulation
- WebSocket-based communication via Chrome DevTools Protocol (CDP)

## Context
This project was significantly simplified (v3.0+) from a full DevTools integration to focus solely on browser preview and multi-instance management, reducing the core codebase by ~85%.
